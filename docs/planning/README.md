# BMALPH Transition Planning Mirror

This directory is the transition-facing BMAD artifact mirror for the current
passkey authentication PR. The canonical planning source remains
`specs/passkey-authentication/`.

The current `bmalph implement` release discovers transition artifacts only from
`_bmad-output/planning-artifacts`, `_bmad-output/planning_artifacts`, or
`docs/planning`. Keeping the active passkey PRD, architecture, epics, and
readiness report here lets the transition command find the implemented feature
without moving the canonical specs bundle.
