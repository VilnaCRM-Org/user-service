Welcome to the Community and Support page for our project! We're glad you're here. This page provides information on how to get involved with the project, where to find help, and how to connect with the community.

## Getting Involved

There are many ways to contribute to our project, from writing code and documentation to reporting bugs and suggesting enhancements. Please see our [Contributing Guide](https://github.com/VilnaCRM-Org/user-service/blob/main/CONTRIBUTING.md) for more details on how to get started.

## Support Channels

- **GitHub Issues**: For reporting bugs or requesting new features, please use our [GitHub Issues](https://github.com/vilnacrm-org/user-service/issues).
- **Branches**: Branches should be based on issues, if your problem does not have issue, please previously create an issue [create branch](https://docs.github.com/en/issues/tracking-your-work-with-issues/creating-a-branch-for-an-issue).
- **Pull Requests**: If you've made improvements to our project, open a [pull request](https://github.com/vilnacrm-org/user-service/pulls) to contribute your changes to the project.

## Reporting Issues

If you encounter a security issue, please report it responsibly. Check our [Security Policy](https://github.com/VilnaCRM-Org/user-service/blob/main/SECURITY.md) for more details on how to report security vulnerabilities.

## Need More Help?

If you've gone through the channels above and still need help, feel free to reach out to the project maintainers directly via [GitHub](https://github.com/vilnacrm-org/user-service). We're here to help!

Thank you for being a part of our community!
